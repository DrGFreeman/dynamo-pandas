Installation
============

.. toctree::
    :maxdepth: 3
    :caption: Contents:


Install ``dynamo-pandas`` from `PyPI <https://pypi.org/project/dynamo-pandas/>`_ using pip:

.. code-block:: console

    $ pip install dynamo-pandas


Requirements
------------

``dynamo-pandas`` has the following requirements:

* ``python`` >= 3.7
* ``boto3``
* ``pandas`` >= 1
